# Next.js + Redux Toolkit + Mantine + TypeScript

SPA со списком карточек [/products](https://vgmaster.github.io/nextjs-simple-spa/products), на каждой из которых выводится картинка и информация.

Используется API [https://openlibrary.org/swagger/docs](https://openlibrary.org/swagger/docs)

## На странице `/products`

- Выводится весь список продуктов
- На карточке иконка лайка. При нажатии на которую ставится или убирается like. Иконка подкрашивается, когда проставлен like
- На карточке иконка удаления. При нажатии на которую карточка удаляется
- Фильтр для просмотра всех карточек и карточек, добавленных в избранное
- Контент карточки (текст) урезан, чтобы у карточек была одинаковая высота
- При клике на любом месте карточки (кроме иконки лайка и кнопки удаления) мы попадаем на отдельную страницу карточки

## На странице `/products/:id`

- Выводится более подробная информация о продукте
- Сделана кнопка для перехода на основную страницу

## На странице `/products/create-product`

- Реализовано создание продукта. Поля обязательные и с минимальной валидацией
- При отправке формы данные сохраняются в общий store

## Реализовано

- Пагинация списка
- Возможность редактирования карточки продукта
- Дополнительная фильтрация
- Поиск (без кнопки отправки)
